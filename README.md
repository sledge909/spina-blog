## Installation
Add this line to your application's Gemfile:

```ruby
gem 'spina-blog'
```

And then execute:
```bash
$ bundle
```

And install:
```bash
$ rails g spina_blog:install
```

Or install it yourself as:
```bash
$ gem install spina-blog
```
